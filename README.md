# space-ayx-notes

一个用于归档和发布独立 HTML 页面的仓库。

## 简介

本仓库主要用于存放各类静态 HTML 页面，内容涵盖学习笔记、工具页面、临时演示页面等。每个页面均为独立文件，不依赖特定域名或后端服务，便于直接查看或分发。

## 目录说明

- `/docs`：主要存放可公开访问的 HTML 页面，可用于 GitHub Pages 等静态托管。
- `/src`：存放页面相关的源文件、素材或未打包的原始代码。
- `/assets`：公共资源文件，如样式表、脚本、图片等。
- `/archive`：已归档或不再维护的页面，保留历史版本。

> 部分目录可能根据实际需要有所调整，具体以仓库当前结构为准。

## 页面归档说明

- 每个 HTML 页面通常自包含，内部引用的资源尽量相对路径或内联。
- 页面可能使用常见前端库（如 Vue、React、Tailwind 等），但一般不依赖复杂构建流程。
- 归档页面保持原样，不保证与最新浏览器完全兼容，但会尽量标注使用的技术栈。
- 页面可能包含临时性内容或实验性功能，使用前请自行评估。

## 维护说明

- 本仓库不定期更新，新增或修改页面会通过 commit 记录说明。
- 欢迎提交 Issue 或 Pull Request 建议改进或新增内容。
- 请勿在本仓库上传包含敏感信息、商业机密或侵犯他人权益的文件。
- 仓库内页面仅供学习、参考或个人使用，请遵守相关法律法规。

## 使用方式

1. 直接打开 `/docs` 目录下的 HTML 文件即可浏览。
2. 若启用 GitHub Pages，可通过仓库设置中的 Pages 功能访问对应页面。
3. 也可将仓库克隆到本地，在浏览器中打开任意 HTML 文件。
